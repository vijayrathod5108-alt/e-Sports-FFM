# 🎮 Elite Esports Tournament - Registration Portal

Yeh ek single-page fully responsive aur modern web registration portal hai jo Esports Tournaments ke liye design kiya gaya hai. Isme smart features integrated hain jaise auto-amount autofill UPI integration aur automatic dynamic WhatsApp detail generator.

---

## 🔥 Features
* 📱 **Fully Responsive UI:** Mobile, Tablet aur Desktop sabhi screen sizes ke liye optimized.
* 💳 **Smart UPI Deep-Linking:** Mobile users jaise hi "Pay" button par click karenge, unke phone ke UPI apps (PhonePe, GPay, Paytm) automatic **₹100** amount ke sath open ho jayenge.
* 📝 **Automated WhatsApp Submission:** Registration form submit karte hi saari details aur charo players ki UIDs ek text format mein auto-compile hokar admin ke WhatsApp number par redirect ho jati hain.
* 📢 **Direct WhatsApp Channel Link:** Isme aapka official channel link pehle se hi added hai taaki players ek click mein join kar sakein.

## 📁 Repository Structure
Aapki repository mein niche di gayi 3 files ka hona compulsory hai:

1. `index.html` — Aapki main website ka working code (UI, Form, Payment integration).
2. `README.md` — Yeh file (Documentation).
3. `1000161207.png` — Aapka PhonePe/GPay QR Code scanner image jo website par display hoga.

> ⚠️ **Zaruri Note:** Apne QR Code photo ka naam badal kar exact `1000161207.png` hi rakhein aur use repository ke main folder mein hi upload karein.

---

## 🛠️ Setup Kaise Karein (GitHub Pages Par Live Karein)

1. **New Repository Banayein:** Apne GitHub account par ek nayi **Public** repository banayein.
2. **Files Upload Karein:** Apni `index.html`, `README.md` aur QR Scanner photo (`1000161207.png`) ko direct main branch mein upload kar dein.
3. **GitHub Pages On Karein:**
   * Repository ke **Settings** tab mein jayein.
   * Left side menu se **Pages** option par click karein.
   * **Build and deployment** section ke andar, *Branch* mein `None` ko badal kar `main` select karein aur **Save** kar dein.
4. **Website Live:** Kuch hi seconds mein GitHub aapko ek live link de dega (Jaise: `https://your-username.github.io/your-repo-name/`).
